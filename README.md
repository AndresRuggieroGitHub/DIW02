# Lexi

Base del proyecto web de Lexi adaptada a la tarea 2 de DIW.

## Descripcion

Lexi es una propuesta de sitio web para practicar ingles con vocabulario personalizado, packs por nivel y una interfaz clara orientada a la usabilidad.

## Contenido principal

- Landing page con hero, dos secciones de contenido y dos franjas destacadas.
- Ficha de producto con maquetacion en grid.
- Pagina de busqueda con resultados estaticos en flexbox.
- Pagina de contacto.
- Pagina de carrito estatico.
- Cabecera comun con logo, navegacion, buscador, selector ES/EN y acceso al carrito.
- Pie comun con contacto, redes y copyright.

## Tecnologias utilizadas

- HTML5
- CSS3 con variables y media queries
- JavaScript para la navegacion movil
- Bootstrap 5
- Bootstrap Icons

## Extras ya integrados

- Tipografia fluida y responsive
- Estados de hover y focus visibles
- Modo oscuro automatico mediante `prefers-color-scheme`

## Ejecucion

1. Descargar o clonar el repositorio.
2. Abrir index.html en el navegador.

Abre [index.html](index.html) en el navegador para visualizar la web.
